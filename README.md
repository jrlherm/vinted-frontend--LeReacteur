## Run the project

Clone this repository :

```console
git clone https://github.com/jrlherm/vinted-frontend--LeReacteur.git
cd vinted-frontend--LeReacteur
```

Install packages :

```console
yarn install
```

When installation is complete, run the project :

```console
yarn dev
```
